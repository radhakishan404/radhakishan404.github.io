01 — Introduction

What Exactly is Google Gemma?

Gemma is a family of open-source, lightweight AI models built by Google DeepMind. Released in early 2024 and updated through 2025–2026, Gemma gives developers and researchers direct access to powerful AI models they can run, modify, and deploy — completely for free.

Think of it as Google giving you a mini version of the same AI technology that powers Gemini — but one you can run on your own laptop, phone, or server without paying anyone a single rupee.

🔑 The simplest way to understand Gemma: It's like having your own private ChatGPT that lives on your computer. No internet. No subscription. No data sharing. Just you and a powerful AI model.

Where does Gemma come from?

Gemma is built by Google DeepMind — the same team behind Gemini (Google's most powerful AI). Gemma uses the same research, architecture, and training techniques as Gemini, but it's scaled down to run efficiently on consumer-grade hardware.

It's released under an open license — meaning developers, students, and companies can use it for both personal and commercial projects without any licensing fees.

Your data never leaves your device. Perfect for sensitive or confidential work.

03 — Model Lineup

Gemma Model Sizes Explained

Gemma comes in different sizes. The number (2B, 7B, 9B, 27B) refers to the number of parameters — basically how "smart" and how "heavy" the model is. More parameters = more capable but needs more RAM.

💡 Which one should you pick? If you have a normal laptop with 8–16GB RAM, start with gemma2:9b via Ollama. It gives the best quality-to-performance ratio for most use cases in 2026.

Instruction-Tuned vs Base Models

Each size comes in two variants. Base model is raw — it just predicts the next word. Instruction-tuned (IT) model is trained to follow instructions and have conversations. For 99% of use cases, always use the instruction-tuned version.

Variant What it does When to use
gemma-2b Predicts text, no conversation ability Research, fine-tuning experiments
gemma-2b-it Follows instructions, chat-ready Chatbots, Q&A, coding help
gemma2-9b-it Latest, smarter instruction following Most production use cases
gemma2-27b-it Highest quality responses Complex reasoning, analysis

05 — Setup Guide

Complete Setup: 3 Ways to Run Gemma

There are three main ways to run Gemma depending on your technical level and use case. We'll cover all three — from the easiest to the most advanced.

Method 1: Ollama (Easiest — Recommended for Beginners)

Ollama lets you run Gemma locally with a single command. It handles everything — downloads, configuration, and serving. This is how 90% of developers run local models.

1

Install Ollama

Visit ollama.com and download for Mac, Windows, or Linux. It's a simple installer — takes 2 minutes.

2

Pull the Gemma Model

Open your terminal and run the command below. It will download the model automatically (around 5–18GB depending on size).

3

Start Chatting

Run the second command below and Gemma will open right in your terminal. Start talking to it immediately.

4

Use the API (Optional)

Ollama also runs a local REST API on port 11434. You can use it to connect Gemma to your own apps.

Terminal — Install & Run
# Step 1: Pull Gemma model (choose your size)
ollama pull gemma2:9b       # Best for most laptops
ollama pull gemma2:2b       # If you have less than 8GB RAM
ollama pull gemma2:27b      # If you have 20GB+ RAM

# Step 2: Run it in terminal (chat mode)
ollama run gemma2:9b

# Step 3: Call via API from your code
curl http://localhost:11434/api/generate -d '{
  "model": "gemma2:9b",
  "prompt": "Explain machine learning in simple words"
}'

Method 2: Google Colab (No Installation Needed)

If you don't want to install anything, Google Colab lets you run Gemma in the cloud for free using Google's GPUs. Perfect for experimentation and learning.

1

Go to Google Colab

Visit colab.research.google.com and create a new notebook. Change runtime to GPU (T4 is free).

2

Accept Gemma License on Kaggle

Go to Kaggle → search "Gemma" → accept the model license. This is required once before using via API.

3

Install and Load the Model

Paste and run the code below in your Colab notebook. The model will load in 2–3 minutes.

Google Colab — Python
# Install required libraries
!pip install -q keras-nlp keras

# Load Gemma model
import keras_nlp

gemma_lm = keras_nlp.models.GemmaCausalLM.from_preset("gemma2_instruct_2b_en")

# Generate a response
response = gemma_lm.generate(
    "What is artificial intelligence? Explain simply.",
    max_length=256
)
print(response)

Method 3: Hugging Face Transformers (For Developers)

For developers who want full control and want to integrate Gemma into Python projects — Hugging Face is the standard approach.

Python — Hugging Face
# Install dependencies
pip install transformers torch accelerate

# ---- Your Python script ----
from transformers import AutoTokenizer, AutoModelForCausalLM
import torch

# Load model and tokenizer from Hugging Face
model_id = "google/gemma-2-9b-it"

tokenizer = AutoTokenizer.from_pretrained(model_id)
model = AutoModelForCausalLM.from_pretrained(
    model_id,
    torch_dtype=torch.bfloat16,
    device_map="auto"
)

# Create a chat message
messages = [
    {"role": "user", "content": "Write a Python function to reverse a string"}
]

# Tokenize and generate
input_ids = tokenizer.apply_chat_template(
    messages, return_tensors="pt", return_dict=True
).to("cuda")

outputs = model.generate(**input_ids, max_new_tokens=256)
response = tokenizer.decode(outputs[0], skip_special_tokens=True)
print(response)

⚠️ Note for Hugging Face: You need to accept the model license on Hugging Face → google/gemma-2-9b-it and log in with huggingface-cli login before downloading.

06 — Getting the Most Out of It

How to Prompt Gemma Effectively

Gemma responds much better when you follow a clear prompting structure. Here are the patterns that work best.

The 3-Part Prompt Formula

Prompt Structure
# ROLE → Tell Gemma who it is
"You are an expert Python developer with 10 years of experience."

# TASK → Tell it exactly what to do
"Write a REST API endpoint using FastAPI that accepts a user's name
and returns a personalised greeting message."

# FORMAT → Tell it how to respond
"Include comments in the code. Keep it under 30 lines. 
Add a brief explanation after the code."

Good vs Bad Prompts

❌ Bad Prompt✅ Good Prompt
"Explain AI" "Explain artificial intelligence to a 15-year-old student in under 100 words with a real-life example."
"Fix my code" "You are a Python expert. Find and fix the bug in this code. Explain what was wrong and why."
"Write an email" "Write a professional follow-up email to a recruiter after an interview. Tone: confident but polite. Length: under 100 words."

10 — Common Questions

Frequently Asked Questions

Is Gemma really free to use commercially?

Yes. Gemma is released under Google's custom open model license which allows commercial use. You can build and sell products using Gemma. Just don't use it to train competing foundation models or use the "Gemma" name in your product name.

How good is Gemma compared to ChatGPT?

Gemma 2 27B comes close to GPT-3.5 level quality on many benchmarks. It won't beat GPT-4 in raw intelligence, but for most practical tasks — coding help, summarization, writing, Q&A — Gemma 9B is surprisingly capable and completely free.

Can I run Gemma on a basic laptop?

Yes! Gemma 2B runs on any laptop with 4GB+ RAM. For Gemma 9B you ideally want 16GB RAM. If your laptop is older or lower spec, use Google Colab (free GPU) instead of running it locally.

Do I need to know Python to use Gemma?

Not for basic usage. With Ollama, you can chat with Gemma directly in your terminal without writing a single line of code. Python is only needed when you want to build apps or integrate Gemma into your own projects.

What languages does Gemma support?

Gemma is primarily trained on English data and performs best in English. It can understand and respond in many other languages including Hindi, but the quality will be lower than English responses.

Can I use Gemma on my phone?

Yes — Gemma 2B is small enough to run on Android devices using Google's MediaPipe or via the Ollama mobile server setup. iOS support is growing but still limited in 2026.
