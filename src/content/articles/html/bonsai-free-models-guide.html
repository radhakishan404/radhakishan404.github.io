01 — Overview

What is Bonsai?

Bonsai kya hai?

Bonsai is a free CLI tool by Boolean Inc. that routes your AI agent's prompts to frontier coding models — GPT-5, Claude Sonnet/Opus, Grok, Gemini Pro/Flash, and Qwen — completely free of charge.

Bonsai ek free CLI tool hai Boolean Inc. ka, jo tumhare AI agent ke prompts ko frontier models pe route karta hai — GPT-5, Claude, Grok, Gemini — bilkul free mein.

The trade-off: Bonsai anonymizes your prompts and uses them for model evaluation and training. You get frontier access, they get anonymized data to improve AI.

Trade-off yeh hai: Bonsai tumhare prompts anonymize karke model training ke liye use karta hai. Tumhe free models, unhe anonymized data.

All data is fully anonymized and encrypted with AES-256. Your identity is never linked to any prompt. Sabka data fully anonymized hai. AES-256 encryption. Tumhari identity kabhi kisi prompt se linked nahi hoti.

03 — Setup guide

Step-by-step setup

Step-by-step setup

1

Apply for access

Access ke liye apply karo

Go to trybons.ai and apply. No credit card needed. Once approved, grab your API key from the dashboard.

trybons.ai pe jao aur apply karo. Credit card nahi chahiye. Approve hone ke baad API key lo.

trybons.ai →
2

Install the Bonsai CLI

Bonsai CLI install karo

Open your terminal and run:

Terminal mein yeh run karo:

npm install -g @bonsai-ai/cli

Requires Node.js. Install from nodejs.org if needed.

Node.js chahiye. nodejs.org se install karo agar nahi hai.

3

Login with your API key

API key se login karo

bonsai login

Paste your API key when prompted. Saved locally — no need to enter again.

Prompt mein API key paste karo. Locally save — dubara nahi maangega.

4

Start the Bonsai proxy

Bonsai proxy start karo

bonsai start

Starts a local proxy. Keep it running in a terminal tab. Your agent routes through this to reach frontier models.

Local proxy start hoti hai. Ek terminal tab mein chalta rakho. Agent isi se frontier models tak pahunchega.

That's it. Three commands and you're live on frontier models.Bas. Teen commands aur setup complete.
04 — Connect to your agent

Connect to Codex / Claude Code

Codex / Claude Code se connect karo

Once bonsai start is running, point your agent to the local proxy endpoint. Authentication and routing happen automatically.

Jab bonsai start chal raha ho, agent ko local proxy endpoint pe point karo. Authentication aur routing automatic hai.

Run bonsai start every time before your agent makes calls — otherwise you'll get a connection refused error. Har baar agent use karne se pehle bonsai start run karo — warna "connection refused" error aayega.

For Claude Code, set the API base URL in ~/.claude/settings.json to Bonsai's local proxy port.

Claude Code ke liye, ~/.claude/settings.json mein API base URL ko Bonsai ke proxy port pe set karo.

05 — Privacy

What happens to my data?

Mera data kahan jaata hai?

Bonsai collects your prompts and completions, fully anonymizes them, and uses them for model evaluation and training. This is the trade-off for free access.

Bonsai tumhare prompts collect karta hai, anonymize karta hai, aur model training ke liye use karta hai. Yahi hai free access ka trade-off.

They do NOT collect your identity, name, or email linked to any prompt or code output.

Tumhari identity, naam, ya email kabhi bhi prompt se linked nahi hoti.

Avoid sending passwords, private keys, or personal data through Bonsai — treat it like any third-party AI service.Passwords, private keys mat bhejo — isko kisi bhi third-party AI service jaisa treat karo.
06 — FAQ

Common questions & errors

Aam sawaal aur errors

Is this actually free forever?Kya yeh sach mein hamesha free hai?
Yes — as long as you're okay with anonymized data being used for training. Model names are hidden but you always get a leading model.Haan — anonymized data use karne se agree ho toh. Model names hidden hain par best model hi milta hai.
Why are model names hidden?Model names kyun hide hain?
Stealth mode enables unbiased evaluation. Knowing the model could influence your prompts and skew training data.Unbiased evaluation ke liye. Agar pata ho kaunsa model hai, prompts aur data dono affect ho sakte hain.
bonsai: command not found
npm global bin path isn't in $PATH. Run npm config get prefix and add that /bin to your ~/.zshrc or ~/.bashrc.npm config get prefix run karo aur us path ka /bin apni shell profile mein add karo.
connection refused
bonsai start isn't running. Start it in a terminal tab before making any agent calls.bonsai start chal nahi raha. Agent calls se pehle terminal mein start karo.
Does it work on Windows?Windows pe kaam karta hai?
Should work via WSL2 or PowerShell. Best results on macOS or Linux.WSL2 ya PowerShell se kaam karna chahiye. Best results macOS/Linux pe.
