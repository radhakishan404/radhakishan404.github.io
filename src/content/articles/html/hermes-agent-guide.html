01 — Overview

What exactly is Hermes Agent?

Hermes Agent is an open-source AI agent built by Nous Research — the same team behind the famous Hermes series of language models. Unlike ChatGPT or Claude which are web chatbots you visit, Hermes is something you install and run yourself on any server or computer.

Think of it like the difference between going to a restaurant vs having a personal chef. ChatGPT is the restaurant — you visit it, it serves you, and when you leave, it forgets you. Hermes is the personal chef who lives in your house, knows your taste, and gets better at cooking for you over time.

It is the only AI agent with a built-in learning loop — meaning it creates skills from experience and improves those skills every time it uses them.

Hermes Agent exactly kya hai?

Hermes Agent ek open-source AI agent hai jo Nous Research ne banaya hai — wahi team jo famous Hermes language models banaati hai. ChatGPT ya Claude jaisi web chatbots ke unlike, Hermes ek cheez hai jo tum khud kisi bhi server ya computer pe install karke run karte ho.

Isko samjhne ka ek aasaan tarika — restaurant vs personal chef. ChatGPT restaurant hai — tum jaate ho, serve karta hai, aur jaate hi bhool jaata hai. Hermes personal chef hai jo tumhare ghar rehta hai, tumhara taste jaanta hai, aur time ke saath aur better hota jaata hai.

Yeh pehla aur abhi tak ka akaila AI agent hai jisme built-in learning loop hai — matlab experience se skills banata hai aur use karne ke saath unhe improve karta rehta hai.

Built by Nous Research — one of the most respected open-source AI labs. Same team that made Hermes 3 and Hermes 4 models used by millions of developers worldwide.
Nous Research ne banaya — ek bohot respected open-source AI lab. Same team jo Hermes 3 aur Hermes 4 models banayi jise duniya bhar ke millions developers use karte hain.
02 — The Need

Why do you actually need this?

Current AI tools have a fundamental problem — they are stateless. Every conversation starts fresh. They don't know your projects, your preferences, or what you worked on yesterday. You are the memory. You paste context every single time.

Tumhe actually yeh kyun chahiye?

Current AI tools ka ek fundamental problem hai — yeh stateless hote hain. Har conversation fresh start hoti hai. Unhe tumhare projects, preferences, ya kal ka kaam pata nahi. Tum hi memory ho. Har baar context paste karte ho.

04 — The Core Idea

The learning loop — simply explained

This is the thing that makes Hermes genuinely different. Every other AI agent gives you the same quality on day 100 as day 1. Hermes does not. Here is exactly how the loop works.

Learning loop — seedha samjho

Yahi cheez Hermes ko genuinely different banati hai. Baaki sab AI agents day 1 aur day 100 mein same quality dete hain. Hermes nahi. Exactly yahan samjho loop kaise kaam karta hai.

1
You give it a task
Tum ek task dete ho

You say something like — "debug this Python error" or "summarize this article every morning." Hermes starts working on it using the tools and skills it already has.

Tum kuch kehte ho jaise — "is Python error ko debug karo" ya "har subah yeh article summarize karo." Hermes apne existing tools aur skills use karke kaam shuru karta hai.

2
It solves the task and remembers how
Task solve karta hai aur yaad rakhta hai kaise

After completing the task, Hermes writes a skill file — a reusable solution packed with the exact steps it used. Next time you ask something similar, it reads that skill first.

Task complete karne ke baad, Hermes ek skill file likhta hai — ek reusable solution jisme exact steps hain jo usne use kiye. Agli baar same cheez poochho, pehle woh skill padhi jaati hai.

3
The skill improves every time it's used
Har use ke saath skill improve hoti hai

Skills are not static. When Hermes uses a skill and finds a better approach, it updates the skill file. So the same task gets done faster and better over time, automatically.

Skills static nahi hain. Jab Hermes skill use karta hai aur better approach milti hai, toh skill file update ho jaati hai. Same task automatically time ke saath faster aur better hota jaata hai.

4
It builds a model of who you are
Yeh tumhare baare mein ek model banata hai

Across all your conversations, Hermes builds up a picture of your preferences, your projects, your working style. It can search its own past conversations to recall context you've long forgotten.

Tumhari saari conversations mein, Hermes tumhari preferences, projects, working style ka ek picture banata hai. Apni past conversations search kar sakta hai un cheezein yaad karne ke liye jo tum bhool gaye ho.

The result: Hermes on day 30 is noticeably better than Hermes on day 1 — and that gap keeps growing. No other free AI agent does this.
Result yeh hai: Day 30 ka Hermes day 1 se clearly better hota hai — aur yeh gap badhta rehta hai. Koi doosra free AI agent yeh nahi karta.
05 — Installation

How to install Hermes Agent

Works on Linux, macOS, WSL2 (Windows), and Android via Termux. One command installs everything.

Hermes Agent kaise install karein

Linux, macOS, WSL2 (Windows), aur Android Termux pe kaam karta hai. Ek command mein sab install ho jaata hai.

1
Run the installer
Installer run karo

Open your terminal and run this single command. It auto-detects your platform and installs everything needed.

Terminal kholo aur yeh ek command run karo. Automatically tumhara platform detect karta hai aur sab kuch install karta hai.

Terminal
curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash
2
Run the setup wizard
Setup wizard run karo

After installation, run the setup wizard. It asks you step by step for your AI provider API key, messaging platform setup (Telegram, etc.), and other preferences. Takes about 3 minutes.

Install ke baad, setup wizard run karo. Yeh step by step poochta hai — AI provider API key, messaging platform setup (Telegram, etc.), aur preferences. Lagbhag 3 minutes lagenge.

Terminal
hermes setup
3
Start Hermes
Hermes start karo

Launch the terminal interface to start chatting immediately. Or run the gateway to connect your messaging apps.

Terminal interface launch karo aur turant baat shuru karo. Ya messaging apps connect karne ke liye gateway run karo.

Terminal — two options
# Option 1: Start terminal chat (simplest way)
hermes

# Option 2: Start messaging gateway (Telegram, Discord etc)
hermes gateway start
4
Alternative — manual install from GitHub
Alternative — GitHub se manual install

If you prefer more control or want to develop on Hermes, you can clone and install manually. Requires Python 3.11.

Agar zyada control chahiye ya Hermes pe develop karna ho, toh manually clone karke install karo. Python 3.11 chahiye.

Git + Python
git clone https://github.com/NousResearch/hermes-agent.git
cd hermes-agent

# Install uv (fast Python package manager)
curl -LsSf https://astral.sh/uv/install.sh | sh

# Create virtual environment and install
uv venv venv --python 3.11
source venv/bin/activate
uv pip install -e ".[all]"

# Now run setup
hermes setup
Windows native is not supported. Use WSL2 on Windows. Android users can use Termux — follow the official Termux guide in the docs.
Windows native supported nahi hai. Windows pe WSL2 use karo. Android users Termux use kar sakte hain — official Termux guide docs mein hai.
06 — First Session

What happens the first time you run it

The setup wizard guides you through everything. Here's what you'll be asked and what to answer.

Pehli baar run karne pe kya hota hai

Setup wizard sab guide karta hai. Yahan bataya hai kya poochha jaayega aur kya answer karna hai.

A
Choose your AI provider
AI provider choose karo

You pick which AI model powers Hermes. The easiest starting options are OpenAI (GPT-4o) or Nous Portal (free tier available). You can change this anytime with hermes model.

Choose karo kaun sa AI model Hermes ko power karega. Start karne ke liye sabse aasaan options hain OpenAI (GPT-4o) ya Nous Portal (free tier available). Kabhi bhi hermes model se change kar sakte ho.

B
Enter your API key
API key daalo

Get your API key from the provider you chose (e.g. platform.openai.com → API keys). Paste it when prompted. Stored securely in ~/.hermes/.env on your machine.

Jo provider choose kiya usse API key lo (jaise platform.openai.com → API keys). Prompt aane par paste karo. Secure ~/.hermes/.env mein store hoti hai tumhare machine pe.

C
Connect Telegram (optional but recommended)
Telegram connect karo (optional but recommended)

Go to @BotFather on Telegram, create a new bot, copy the bot token. Paste it in setup. Then you can talk to Hermes from your phone anywhere in the world.

Telegram pe @BotFather pe jao, naya bot banao, bot token copy karo. Setup mein paste karo. Phir duniya mein kahin se bhi phone se Hermes se baat kar sakte ho.

D
Start your first conversation
Pehli conversation shuru karo

Just start talking naturally. Try: "What can you do?" to see all capabilities. Or ask it to do something real — search the web, write a script, set a daily reminder. The more you use it, the smarter it gets.

Naturally baat shuru karo. Try karo: "What can you do?" sab capabilities dekhne ke liye. Ya kuch real karne kaho — web search, script likhna, daily reminder set karna. Jitna use karoge, utna smart hoga.
