RkCodex Auto DM

Privacy Policy

Effective date: April 9, 2026

This Privacy Policy explains how the RkCodex Auto DM application collects, uses, and protects information when used with Instagram messaging and comment-management features for testing and account automation workflows.

Information We Access

The application may access the following information from Instagram and Meta APIs:

  • Instagram professional account identifiers and profile information
  • Comments posted on your Instagram content
  • Direct messages and messaging metadata needed to respond to users
  • Webhook event payloads required to process comment and messaging events
  • Access tokens and technical configuration required for API authentication

How We Use Information

The application uses this information only to:

  • Detect comments that match configured trigger keywords
  • Send private replies or direct messages through Instagram APIs
  • Process replies from users who interact with the account
  • Operate, debug, and monitor the automation workflow
  • Maintain basic logs needed for troubleshooting and delivery verification

Data Sharing

We do not sell personal data. Data accessed through the application is used only for the automation and testing purposes described in this policy, unless required by law or platform policy.

Data Retention

Access tokens, webhook payloads, and related operational data may be retained only as long as needed to operate, troubleshoot, or secure the application. Data that is no longer needed should be deleted or rotated promptly.

Security

Reasonable technical and organizational measures are used to protect access tokens, webhook payloads, and account-related data. However, no internet transmission or storage system can be guaranteed to be fully secure.

User Rights and Data Deletion

If you want your information removed from this application or want to revoke access, you may:

  • Remove the application from Instagram Apps and Websites settings
  • Revoke Meta access tokens connected to the application
  • Request deletion by contacting the email listed below
